## Athletics Intelligence Landing Page

## Running app Locally

```bash
npm install
```

```bash
npm run dev
```

Then application should be accessible from `http://localhost:4321`


## Running app using Docker

```bash
docker build --no-cache -t skn-athletics-intelligence-landing-page .
```

```bash
docker run --rm -p 4321:4321 skn-athletics-intelligence
```
